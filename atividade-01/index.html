Pizzaria

criação de um inteface para um site de uma pizzaria

A proposta é criar uma interface para um site de uma pizzaria, contendo as linguagens de HTML e CSS.

Espero que com esta atividade eu poça aprender o básico de como posso criar um projeto de interface

    
    # Código de Exemplo: Pedido de Pizza

    def fazer_pedido(sabor, tamanho, endereco, telefone):
        print("Pedido feito com sucesso!")
        print("Pizza:", sabor)
        print("Tamanho:", tamanho)
        print("Endereço de Entrega:", endereco)
        print("Telefone para Contato:", telefone)

    # Exemplo de uso
    fazer_pedido("Calabresa", "Grande", "Rua das Pizzas, 123", "55 1234-5678")
